# 網站品質檢查報告

## 已知限制

- 本站是純靜態示範網站，沒有後端、資料庫、會員、購物車、金流或即時預約排程。
- `DEMO_MODE = true` 時所有聯絡行為只顯示提示，不會真正聯絡品牌。
- 作品區目前以 CSS 自製圖形呈現，不代表真實作品照片。
- 課程時間均清楚標示為示範資訊，不能直接作為真實招生承諾。
- 沒有加入流量分析、廣告追蹤或行銷 Cookie。
- 沒有虛構地址，因此目前不顯示地圖。
- 字體使用系統內建的繁體中文字型，不另外下載大型網頁字型。
